# Xelly Points Investor Landing Page

## Objective

Initialize a maintainable, investor-facing landing page for a discount app. The first pass is presentation-focused and should give a future coding agent a clear, modern foundation to extend.

## Technical approach

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
- No backend or database in the initial scaffold

## Initial structure

The application should be organized around focused landing-page sections: hero, product value, market opportunity, traction, investment thesis, and final call to action. Components should remain easy to replace or expand as product facts and brand direction become available.

## Success criteria

- The project is runnable from the workspace root.
- The app has a clear component and styling structure for a single-page investor narrative.
- A future coding agent can begin implementing the visual design without reworking the project foundation.
- The setup passes the available lint/build checks.

## Scope boundary

This initialization does not invent claims about users, revenue, market size, or traction. Those will be supplied later as product inputs.
